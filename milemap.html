Web Application for Task Management

Milemap

UX/UI Designer

January - April 2023

Designed and developed 'Milemap,' a web application for task management that enables users to create, modify, customize, search, browse, and track roadmaps tailored to their preferences. The application also offers features like cloning, deleting, and editing roadmaps for self-improvement, helping to prevent procrastination. Using Agile methodology, FastAPI and Python were utilized for the backend, while SQL and PostgreSQL was employed for database management.

I was responsible for UX/UI design, conducting user research, designing the user interface, and creating user flows and diagrams for each page to ensure a smooth and user-friendly experience.
