Ebay Data Warehouse Design

Data Warehouse for eBay Sales and Order Life Cycle Analysis and Cost Estimation

March - May 2024

Designed and developed a data warehouse for eBay order lifecycle and sales data, implementing a full data pipeline on AWS. The process involved using Amazon S3 for data storage and RDS for relational database management, followed by utilizing AWS Glue for ETL tasks. A BI dashboard was created using AWS QuickSight to analyze sales and track order lifecycles.

The first step in the process was loading the source data into the database and designing the data warehouse following a four-step approach. This included defining Slowly Changing Dimensions (SCD) and creating a confirmation matrix with two fact tables: eBay order lifecycle and eBay sales data. The next step involved loading the dimensions and building the ETL pipeline using AWS Glue.

4 Steps Design

Lastly, created a BI dashboard in AWS QuickSight for analysis, focusing on laptop sales and the order lifecycle.
